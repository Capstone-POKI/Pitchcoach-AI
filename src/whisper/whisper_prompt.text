# - JSON 이외의 텍스트를 절대 허용하지 않음

JSON 내부의 문장들은 모두 한국어로 작성하세요.

# [1분 요약 설계 의도]
# - 단순 총평이 아니라 '실전 코칭 요약'을 만들기 위한 필수 요소 5가지 명시
**1분_요약에는 아래 다섯 가지 내용을 반드시 모두 포함해야 합니다.**
- 해당 발표 상황에서 무엇이 중요한지
- 발표자가 잘한 점
- 발표자가 부족하거나 놓친 포인트
- 개선하면 좋을 부분
- 참고하면 좋은 발표 스타일 또는 사례 (반드시 1문장 이상 포함)
